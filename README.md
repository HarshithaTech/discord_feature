# EduSphere Discord Clone

EduSphere is a premium, feature-rich Discord clone built with modern web technologies, designed for seamless communication and collaboration.

## 🚀 Features

- **Real-time Messaging**: Instant communication with friends and community members.
- **Server Management**: Create, customize, and manage your own servers and channels.
- **Voice & Video Calls**: High-quality voice and video meetings powered by LiveKit.
- **Dynamic UI**: A professional, dark-themed interface with smooth animations and responsive design.
- **Whiteboard Integration**: Collaborative whiteboard for brainstorming and teaching.
- **Authentication & Database**: Secure user authentication and data management using Firebase and Supabase.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4, Framer Motion (for animations)
- **Real-time Communication**: LiveKit
- **Backend Services**: 
  - **Firebase**: Authentication, Firestore (Real-time DB), Storage
  - **Supabase**: PostgreSQL database management
- **Icons**: Lucide React

## 📦 Getting Started

### Prerequisites

- Node.js (latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/HarshithaTech/discord_feature.git
   cd discord_feature
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment Variables:
   Create a `.env` file in the root directory and add your configuration keys (see `.env.example` for reference).

4. Run the development server:
   ```bash
   npm run dev
   ```
